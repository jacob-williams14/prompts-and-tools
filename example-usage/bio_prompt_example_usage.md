# Example Usage - Biography Generation

## How to Use the Parameters Section

1. **Fill in the parameters at the top** with your specific values
2. **Copy the usage template** and reference your defined parameters
3. **The template will use your parameter values** for a personalized biography

## Example: Jacob's Professional Biography

### Step 1: Define Parameters
```
DEVELOPER_NAME: Jacob Williams
EXPERIENCE_LEVEL: Mid-level Developer transitioning to Technical Leadership roles
CAREER_ASPIRATIONS: Technical Leadership, System Architecture, Team Mentoring, Cross-functional Project Leadership

OPTIONAL_OUTPUT_DIRECTORY:
professional-bios

PERSONAL_DESCRIPTION: 
Jacob is a passionate full-stack developer who thrives on solving complex technical challenges and building scalable solutions. He brings a collaborative approach to development work and is energized by opportunities to mentor team members and drive architectural decisions.

PROJECT_SUMMARIES:
- Root Leading Change Project (project-experience-summaries/root-leading-change-project-summary.md)
- Root Compass Project (project-experience-summaries/root-compass-project-summary.md)

OPTIONAL_STRENGTHS:
Strategic Thinking, Problem-Solving, Collaboration, Technical Innovation, Quality Focus
```

### Step 2: Use the Template
```
Please create a professional biography for Jacob Williams' Atomic Object employee profile following the Professional Biography Generation framework.

Career Context: Mid-level Developer transitioning to Technical Leadership roles with aspirations toward Technical Leadership, System Architecture, Team Mentoring, Cross-functional Project Leadership.

Jacob is a passionate full-stack developer who thrives on solving complex technical challenges and building scalable solutions. He brings a collaborative approach to development work and is energized by opportunities to mentor team members and drive architectural decisions.

Please analyze these project experience summaries:
- Root Leading Change Project (project-experience-summaries/root-leading-change-project-summary.md)
- Root Compass Project (project-experience-summaries/root-compass-project-summary.md)

Strengths: Strategic Thinking, Problem-Solving, Collaboration, Technical Innovation, Quality Focus

Output Directory: professional-bios
Generate a compelling 300-500 word professional biography suitable for potential clients, business executives, and tech recruiters.
```

## Benefits of Using Parameters

- ✅ **Define once, use everywhere** - No need to repeat personal info
- ✅ **Easy to update** - Change your context in one place
- ✅ **Less error-prone** - Reduces copy/paste mistakes
- ✅ **Team-friendly** - Clear structure for others to follow
- ✅ **Quick setup** - Just fill in your parameters and go

## For Team Members

Each person can create their own parameter block:
```
DEVELOPER_NAME: [Your Name]
EXPERIENCE_LEVEL: [Your current level]
CAREER_ASPIRATIONS: [Your goals]
OPTIONAL_OUTPUT_DIRECTORY: [Your custom directory or leave blank for default]
PERSONAL_DESCRIPTION: [Your professional identity/passion]
PROJECT_SUMMARIES: [Your project files]
OPTIONAL_STRENGTHS: [StrengthsFinder themes or leave blank]
```

Then use the usage template with your parameter values!
